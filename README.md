# How to Use

```
./shell-lock-cli -h
NAME:
   shell-lock-cli - CLI Tool to run shell-lock operations

USAGE:
   shell-lock-cli [global options] command [command options]

VERSION:
   0.1.0

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --command value    The shell command to execute
   --lock-file value  The mutex lock file
   --try-lock         try to acquire the lock without waiting (default: false)
   --bash-path value  path to bash executable, the default is /bin/bash or C:\Program Files\Git\bin\bash.exe on Windows (default: /bin/bash)
   --help, -h         show help
   --version, -v      print the version
```
